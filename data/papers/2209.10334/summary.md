# 논문 메타
- 제목: Trade Co-occurrence, Trade Flow Decomposition, and Conditional Order Imbalance in Equity Markets
- 저자: Yutong Lu, Gesine Reinert, Mihai Cucuringu
- arXiv ID: 2209.10334
- 공개일: 2022-09-21
- 요약 근거: abstract 기반 요약 (PDF 저장 성공, 본문 자동 파싱은 미실시)

# 문제 정의 및 배경
체결 간 시간적 근접성(time proximity)이 단기 가격 형성에 유의미한 정보를 담는지, 그리고 이를 단순한 imbalance 신호로 분해해 전략화할 수 있는지 검증한다.

# 핵심 가설/아이디어
모든 체결을 한 묶음으로 보지 않고, 체결 동시성(co-occurrence) 유형별로 나눈 뒤 조건부 불균형(COI)을 계산하면 미래 수익률 예측력이 개선된다는 가설이다.

# 데이터
- 데이터 소스: 주식 시장 체결 데이터
- 기간: 4년
- 샘플 구성: 457개 종목, 체결 이벤트를 5개 유형으로 분류
- 피처: 유형별 정규화 order imbalance(COI), 동시 수익률/미래 수익률

# 방법론
- 모델/알고리즘: 체결 흐름 분해 + 조건부 불균형 지표(COI) 설계
- 학습/검증 절차: 동시 상관 및 미래 예측 연관 분석 + 전략 백테스트
- 하이퍼파라미터: 분류 시간창/정규화 상세는 abstract 기준 제한적

# 실험/결과
- COI와 동시 수익률 간 강한 양(+)의 상관을 확인
- 타 종목 체결과 분리된 고립형 체결 COI는 미래 수익률에 양(+) 연관, 반대 유형은 음(-) 연관
- COI 기반 전략이 비교적 높은 수익/샤프를 기록했다고 보고

# 재현성
- 코드/데이터 공개 여부: abstract 기준 명시 없음
- 구현 난이도: 중간 (체결 분류 로직 + 지표 계산 파이프라인 필요)
- 필요한 의존성: 체결 데이터 정렬/정합, trade-sign 규칙, 백테스트 엔진

# 한계 및 실패 사례
- 일별(daily) 실험 결과라 초단기 실행 리스크(슬리피지/지연) 반영은 제한적
- 종목군/시장 구조에 따라 co-occurrence 신호가 약화될 가능성
- 신호 분해 단계가 복잡해질수록 과최적화 위험이 증가

# 확장 아이디어
- 단일 종목 intraday로 축소해 "체결 간격 기반 2-3개 신호"만 남긴 경량 버전 구현
- COI 신호에 거래비용 임계치(cost gate)를 붙여 과거대응 최소화
- 크립토 체결 데이터로 이식해 체결 동시성 패턴의 시장 간 공통성 검증
