# 문제 정의 및 배경
주문흐름(order flow) 신호의 예측력은 모델 복잡도보다 정규화(normalization) 방식에 크게 좌우될 수 있다. 어떤 정규화가 정보성 신호를 보존하는지가 핵심 문제다.

# 핵심 가설/아이디어
시가총액 정규화가 informed trading 신호에 대한 matched filter 역할을 하며, 거래대금 정규화 대비 미래 수익률 설명력을 크게 개선한다는 가설을 제시한다.

# 데이터
- 소스: 한국 주식시장 데이터(abstract 기준)
- 기간/샘플: abstract에서 상세 범위 미공개
- 피처: 주문흐름 강도 및 정규화 변형(시가총액, 거래대금 등)

# 방법론
- 이론: 신호처리 관점의 정규화 재해석
- 검증: 이론 모델 + 몬테카를로 시뮬레이션 + 실증 검증
- 비교: 시가총액 정규화 vs 전통적 거래대금 정규화

# 실험/결과
- 시가총액 정규화가 미래 수익률과의 상관에서 약 1.32~1.97배 개선
- 설명력(explanatory power) 482% 개선 보고
- 핵심 메커니즘: 거래대금 정규화는 inverse turnover 변동성으로 신호를 오염시킬 수 있음

# 재현성
