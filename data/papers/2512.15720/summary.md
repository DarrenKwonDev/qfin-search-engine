- 공개일: 2025-12-02
- 요약 근거: abstract 기반 요약 (PDF는 저장했지만 본문 섹션 파싱은 미실시)

# 문제 정의 및 배경
고빈도 시장에서 가격 방향 예측은 어려운 반면 변동성(절대 수익률 크기)은 예측 가능한지 검증한다. 핵심은 주문흐름의 정보량을 엔트로피로 측정하는 접근이다.

# 핵심 가설/아이디어
order-flow entropy는 방향(sign) 정보는 제공하지 않지만, 향후 단기 절대수익률 크기(변동성 상태)는 유의미하게 예측할 수 있다는 가설이다.

# 데이터
- 소스: SPY 체결 데이터
- 규모: 38.5M trades
- 기간/구성: 36 거래일, 초 단위 해상도
- 피처: 15-state Markov transition 기반 실시간 order-flow entropy

# 방법론
- entropy 하위 5% 구간 조건부 분석
- 5분 후 절대수익률 반응 측정
- walk-forward 검증 + label-permutation placebo 테스트

# 실험/결과
- entropy 하위 5%에서 이후 5분 절대수익률이 약 2.89배 증가
- 방향 정확도는 45%로 유의하지 않아, 방향성과 변동성 예측이 분리됨을 확인
- out-of-sample 및 placebo 검증에서 통계적으로 강한 신호 제시

# 재현성
- 코드/데이터 공개 여부: abstract 직접 명시 없음
- 구현 난이도: 중간 (초단위 체결 데이터와 상태전이 계산 필요)
- 필요한 의존성: 마코프 전이/엔트로피 계산, 이벤트 기반 백테스트

# 한계 및 실패 사례
- 단일 자산(SPY), 짧은 기간(36일)으로 일반화 한계
- 방향성 알파를 직접 제공하지 않아 단독 방향 전략으로는 제한적
- 거래비용/슬리피지 포함 시 실전 수익성 재검증 필요

# 확장 아이디어
- 초심자 전략: entropy를 변동성 필터로 사용해 진입 시점만 선별(방향은 별도 단순 신호와 결합)
- 다자산/장기 구간으로 안정성 테스트
- 변동성 예측 신호를 포지션 크기 조절(risk scaling)에 활용
